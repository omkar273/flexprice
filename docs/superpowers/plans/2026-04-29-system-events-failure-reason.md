# System Events `failure_reason` Column Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `failure_reason TEXT` column to `system_events` and write to it whenever webhook delivery fails, so the cause is always visible on the row without digging through logs.

**Architecture:** Add `failure_reason` to the Ent schema, regenerate Ent code, add a raw SQL migration, implement `OnFailed(ctx, id, reason)` on the repository, then call it at both failure points: `absorbDeliveryError` in the Kafka handler (primary) and `RetryStalePendingWebhooks` in the webhook service (secondary). Success paths never touch the column.

**Tech Stack:** Go 1.23, Ent ORM, PostgreSQL, Watermill (Kafka consumer), Temporal

## Task 3: Implement `OnFailed` on the repository

**Files:**

- [ ] **Step 1: Add the `OnFailed` method**

Add after the `OnDelivered` method (line ~123) in `internal/repository/ent/systemevent.go`:

```go
// OnFailed records the reason a webhook delivery failed. It overwrites any
// previous failure_reason and never clears it — not even on later success.
func (r *SystemEventRepository) OnFailed(ctx context.Context, eventID, reason string) error {
	if eventID == "" {
		return nil
	}
	return r.client.Writer(ctx).SystemEvent.UpdateOneID(eventID).
		SetUpdatedAt(time.Now().UTC()).
		SetFailureReason(reason).
		Exec(ctx)
}
```

Replace the existing `absorbDeliveryError` method with:

```go
// absorbDeliveryError logs delivery failures for the Kafka consumer path and always acks.
// It also persists the failure reason on the system_events row so it is never silently dropped.
func (h *handler) absorbDeliveryError(ctx context.Context, transport string, err error, event *types.WebhookEvent, messageUUID string) {
	if err == nil {
		return
	}
	if webhookMissingDataError(err) {
		h.logger.Errorw("skipping webhook; referenced data not found (ack, no retry)",
			"transport", transport,
			"error", err,
			"message_uuid", messageUUID,
			"event_name", event.EventName,
			"tenant_id", event.TenantID,
		)
	} else {
		h.logger.Errorw("failed to send webhook",
			"transport", transport,
			"error", err,
			"message_uuid", messageUUID,
			"tenant_id", event.TenantID,
			"event", event.EventName,
		)
	}
	if h.systemEventRepo != nil && event.ID != "" {
		if dbErr := h.systemEventRepo.OnFailed(ctx, event.ID, err.Error()); dbErr != nil {
			h.logger.Warnw("failed to persist webhook failure_reason",
				"error", dbErr,
				"event_id", event.ID,
			)
		}
	}
}
```
